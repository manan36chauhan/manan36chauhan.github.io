Pharmaceutical Delivery Project
Using React-Native

I developed the SmartRXHub application, a delivery management system designed to streamline pharmaceutical logistics for Android and iOS platforms. Built using React Native, this app showcases my cross-platform development skills. Key features include:

- An "Online Status" toggle to manage availability.
- A delivery list with detailed pick-up information.
- Integrated map functionality for navigation and tracking.
- Interactive swipe actions to accept or reject deliveries.

This project highlights my expertise in creating efficient, user-friendly applications for both Android and iOS.
